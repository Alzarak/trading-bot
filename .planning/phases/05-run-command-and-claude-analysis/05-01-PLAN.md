       @dataclass
       class ClaudeRecommendation:
           symbol: str
           action: Literal["BUY", "SELL", "HOLD"]
           confidence: float
           reasoning: str
           strategy: str
           atr: float
           stop_price: float

           def to_signal(self) -> Signal:
               return Signal(
                   action=self.action,
                   confidence=self.confidence,
                   symbol=self.symbol,

                   stop_price=self.stop_price,

       - __init__(self, config: dict, confidence_threshold: float = 0.6)
       - build_analysis_prompt(self, symbol: str, df: pd.DataFrame, strategy_name: str) -> str:
         Takes indicator-enriched DataFrame from MarketScanner.scan().
         Extracts the last 5 rows of key indicator columns (using get_indicator_columns() naming).
         Builds a prompt that:
           a) Shows the symbol and strategy context
           b) Includes a table of the last 5 OHLCV + indicator rows as text
           c) Specifies the JSON response schema: {"symbol": str, "action": "BUY"|"SELL"|"HOLD", "confidence": 0.0-1.0, "reasoning": str, "strategy": str, "atr": float, "stop_price": float}
           d) Instructs Claude to analyze the indicators and return ONE JSON object (not an array)
           e) Emphasizes: "You are an analyst only. Return a recommendation. Do NOT execute trades."
       - parse_response(self, response_text: str) -> list[ClaudeRecommendation]:
         Extracts JSON from response text (handles markdown code blocks).
         Validates required fields exist.
         Filters by confidence_threshold.
         Returns list of ClaudeRecommendation (typically 0 or 1 items).
         On parse failure, logs warning via loguru and returns [].

    IMPORTANT: ClaudeAnalyzer does NOT call Claude itself. It builds prompts and parses responses. The /run command (or bot.py agent mode) is responsible for sending the prompt to Claude and receiving the response. This separation keeps ClaudeAnalyzer fully testable without mocking LLM calls.
