    Create `scripts/risk_manager.py` implementing the `RiskManager` class. Follow the architecture pattern from the research document exactly:

    ```python
    """Risk management module for trading bot.

    Enforces circuit breaker, position sizing, PDT tracking, max position limits,
    and exponential backoff retry with ghost position prevention.
    """
    import json
    import math
    import os
    import time
    import uuid
    from datetime import datetime, timedelta
    from pathlib import Path

    from loguru import logger

    # Import conditionally to allow testing without alpaca-py installed
    try:
        from alpaca.common.exceptions import APIError
    except ImportError:
        APIError = Exception
    ```

    Class structure:
    - `__init__(self, config: dict, trading_client)` — store config and client, set `circuit_breaker_triggered = False`, `start_equity = None`, load PDT trades from JSON file
    - `initialize_session(self)` — check for circuit_breaker.flag in CLAUDE_PLUGIN_DATA, raise RuntimeError if found; else capture start_equity from `client.get_account().equity`
    - `check_circuit_breaker(self) -> bool` — return True if already triggered; compute loss_pct = (start_equity - current_equity) / start_equity * 100; if >= max_daily_loss_pct, set flag, persist to file, log critical, return True; else return False
    - `_persist_circuit_breaker(self)` — write "triggered" to `CLAUDE_PLUGIN_DATA/circuit_breaker.flag`
    - `calculate_position_size(self, symbol: str, current_price: float, size_override_pct: float | None = None) -> int` — if size_override_pct is not None, clamp to [max_position_pct * 0.5, max_position_pct * 1.5]; compute position_value = equity * (pct / 100); cap at budget_usd; return floor(position_value / current_price), or 0 if < 1
    - `check_position_count(self) -> bool` — return len(positions) < max_positions (default 10)
    - `check_pdt_limit(self, symbol: str, date: str) -> str` — filter _pdt_trades to last 7 calendar days; count >= 3 -> "block", count == 2 -> "warn", else "allow"
    - `record_day_trade(self, symbol: str, date: str)` — append to _pdt_trades list and write to pdt_trades.json
    - `_load_pdt_trades(self) -> list[dict]` — read from CLAUDE_PLUGIN_DATA/pdt_trades.json if exists
    - `_save_pdt_trades(self)` — write _pdt_trades to pdt_trades.json
    - `submit_with_retry(self, request, symbol: str) -> object | None` — loop up to 5 attempts; on APIError with 422/403 return None immediately; before retry check get_open_position(symbol) for ghost; sleep [1,2,4,8] seconds between retries; return None after all attempts fail

    Use `logger` from loguru for all logging (info, warning, error, critical).
    Use `os.environ.get("CLAUDE_PLUGIN_DATA", "/tmp")` for file paths so tests can override.

    Run `python -m pytest tests/test_risk_manager.py -x -q` after implementation. All tests must pass GREEN.
