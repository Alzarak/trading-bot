Create `scripts/bot.py` as the main entry point for the trading loop:

1. **Imports:** Import all Phase 3 modules (MarketScanner, OrderExecutor, StateStore, PortfolioTracker, RiskManager, STRATEGY_REGISTRY, Signal). Import APScheduler, signal module, json, os, sys, pathlib, loguru, zoneinfo.

2. **Module-level globals:**
   - `_shutdown_requested = False`
   - `ET = ZoneInfo("America/New_York")`

3. **Signal handlers:**
```python
import signal as signal_module

def _handle_shutdown(signum, frame):
    global _shutdown_requested
    logger.info("Shutdown signal received ({}). Finishing current cycle...", signum)
    _shutdown_requested = True

signal_module.signal(signal_module.SIGINT, _handle_shutdown)
signal_module.signal(signal_module.SIGTERM, _handle_shutdown)
```

4. **`load_config() -> dict`** — reads `config.json` from `CLAUDE_PLUGIN_DATA` directory. Falls back to current directory. Returns parsed JSON dict.

5. **`create_clients(config: dict) -> tuple`** — creates and returns `(TradingClient, StockHistoricalDataClient)` from alpaca-py. Uses `ALPACA_API_KEY` and `ALPACA_SECRET_KEY` env vars. Sets `paper=config.get("paper_trading", True)`. Import alpaca conditionally for testability.

6. **`scan_and_trade(scanner, strategies, executor, tracker, state_store, config)`** — the core pipeline function:
   - If `_shutdown_requested`, return immediately
   - If not `scanner.is_market_open()`, log "Market closed — skipping" and return
   - For each symbol in `config["watchlist"]`:
     - `df = scanner.scan(symbol)` — fetch bars + compute indicators
     - If df is empty, log warning and skip symbol
     - For each strategy_config in `config["strategies"]`:
       - Get strategy class from `STRATEGY_REGISTRY[strategy_config["name"]]`
       - Instantiate and call `strategy.generate_signal(df, symbol, strategy_config.get("params", {}))`
       - If signal.action == "BUY":
         - `current_price = df.iloc[-1]["close"]`
         - `order = executor.execute_signal(signal, current_price)`
         - If order is not None:
           - `state_store.upsert_position(symbol, ..., strategy=signal.strategy)`
           - `tracker.log_trade(symbol, "BUY", current_price, qty, signal.strategy, "bracket")`
       - If signal.action == "SELL":
         - `order = executor.execute_signal(signal, current_price)`
         - If order is not None:
           - `state_store.mark_position_closed(symbol)`
           - Compute P&L from entry price vs current price
           - `tracker.log_trade(symbol, "SELL", current_price, qty, signal.strategy, "market", pnl=pnl)`

7. **`perform_graceful_shutdown(trading_client, state_store)`:**
   - Log "Closing all open positions..."
   - `positions = trading_client.get_all_positions()`
   - For each position: `trading_client.close_position(pos.symbol)`, `state_store.mark_position_closed(pos.symbol)`, log success
   - Catch exceptions per position, log errors
   - Log "Graceful shutdown complete."

8. **`main()`:**
   - Load config
   - Create clients
   - Initialize StateStore at `CLAUDE_PLUGIN_DATA/trading.db`
   - Run crash recovery: `state_store.reconcile_positions(trading_client)`
   - Create RiskManager with `state_store=state_store`
   - Call `risk_manager.initialize_session()`
   - Create MarketScanner, OrderExecutor, PortfolioTracker
   - Load strategies from config
   - Create `BackgroundScheduler(timezone="America/New_York")`
   - Add job with `IntervalTrigger(seconds=60)`, `misfire_grace_time=30`, `coalesce=True`
   - `scheduler.start()`
   - Log "Trading bot started. Press Ctrl+C to stop."
   - Wait loop: `while not _shutdown_requested: time.sleep(1)`
   - `scheduler.shutdown(wait=True)`
   - `perform_graceful_shutdown(trading_client, state_store)`
   - `state_store.close()`
   - Log final P&L summary

9. **`if __name__ == "__main__": main()`**
