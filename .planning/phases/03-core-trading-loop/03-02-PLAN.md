```python
class RiskManager:
    def submit_with_retry(self, request, symbol: str) -> object | None:
        """Submit order with exponential backoff. Returns Order on success, None on failure."""
    def calculate_position_size(self, symbol: str, current_price: float, size_override_pct: float | None = None) -> int:
        """Returns whole shares count (0 if below 1 share)."""
    def check_circuit_breaker(self) -> bool:
        """True if circuit breaker is triggered (trading halted)."""
    def check_pdt_limit(self, symbol: str, date: str) -> str:
        """Returns 'allow', 'warn', or 'block'."""
    def check_position_count(self) -> bool:
        """True if new position can be opened (under limit)."""
```

Create `scripts/order_executor.py` with an `OrderExecutor` class:

1. `__init__(self, risk_manager: RiskManager, config: dict)` — stores risk_manager and config. Extract `atr_multiplier` from `config.get("atr_multiplier", 1.5)`.

2. `calculate_stop_price(self, entry_price: float, atr: float, side: str = "buy") -> float` — computes ATR-based stop: for buy orders, `stop = round(entry_price - (atr * self.atr_multiplier), 2)`. For sell (short — future), `stop = round(entry_price + (atr * self.atr_multiplier), 2)`. Always round to 2 decimal places (Alpaca rejects fractional prices). Returns the rounded stop price.

3. `calculate_take_profit_price(self, entry_price: float, atr: float, risk_reward_ratio: float = 2.0) -> float` — computes `round(entry_price + (atr * self.atr_multiplier * risk_reward_ratio), 2)`.

4. `submit_market_order(self, symbol: str, qty: int, side: OrderSide) -> object | None` — creates `MarketOrderRequest(symbol=symbol, qty=qty, side=side, time_in_force=TimeInForce.DAY)`. Passes to `self.risk_manager.submit_with_retry(request, symbol)`. Returns order or None.

5. `submit_limit_order(self, symbol: str, qty: int, limit_price: float, side: OrderSide) -> object | None` — creates `LimitOrderRequest(symbol=symbol, qty=qty, side=side, time_in_force=TimeInForce.DAY, limit_price=round(limit_price, 2))`. Passes to submit_with_retry.

6. `submit_bracket_order(self, symbol: str, qty: int, limit_price: float, atr: float, side: OrderSide = OrderSide.BUY) -> object | None` — creates a bracket order using `LimitOrderRequest` with `order_class=OrderClass.BRACKET`, `take_profit=TakeProfitRequest(limit_price=self.calculate_take_profit_price(limit_price, atr))`, `stop_loss=StopLossRequest(stop_price=self.calculate_stop_price(limit_price, atr))`. All prices rounded to 2 decimals.

7. `submit_trailing_stop(self, symbol: str, qty: int, trail_percent: float, side: OrderSide = OrderSide.SELL) -> object | None` — creates `TrailingStopOrderRequest(symbol=symbol, qty=qty, side=side, time_in_force=TimeInForce.GTC, trail_percent=round(trail_percent, 2))`. Passes to submit_with_retry.

8. `execute_signal(self, signal: Signal, current_price: float) -> object | None` — the main entry point. Takes a Signal dataclass and current price. Runs all risk checks in order:
   - `risk_manager.check_circuit_breaker()` — if True, log and return None
   - `risk_manager.check_position_count()` — if False, log and return None
   - `risk_manager.check_pdt_limit(signal.symbol, today_str)` — if 'block', return None
   - `risk_manager.calculate_position_size(signal.symbol, current_price)` — if 0, return None
   - For BUY signals: submit bracket order with ATR-based stops
   - For SELL signals: submit market order to close position
   - Log the order result
   - Return the order object or None

Import alpaca-py conditionally (try/except) like risk_manager.py. Use loguru for logging. Import Signal from `scripts.types`.
