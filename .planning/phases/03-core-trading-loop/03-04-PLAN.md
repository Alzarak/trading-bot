**`scripts/strategies/momentum.py` — MomentumStrategy (STRAT-01):**
- `generate_signal(self, df, symbol, params)`:
  - Read last row: `row = df.iloc[-1]`
  - Read previous row: `prev = df.iloc[-2]` (for crossover detection)
  - Derive column names programmatically: `rsi_col = f"RSI_{params.get('rsi_period', 14)}"`, `macd_h_col = f"MACDh_{params.get('macd_fast', 12)}_{params.get('macd_slow', 26)}_{params.get('macd_signal', 9)}"`, `ema_short_col = f"EMA_{params.get('ema_short', 9)}"`, `ema_long_col = f"EMA_{params.get('ema_long', 21)}"`, `atr_col = f"ATRr_{params.get('atr_period', 14)}"`
  - Check NaN on critical values: if any are NaN, return HOLD with reasoning "insufficient data"
  - **BUY conditions** (all must be true):
    1. RSI crosses above 30: `row[rsi_col] > 30 and prev[rsi_col] <= 30`
    2. MACD histogram positive: `row[macd_h_col] > 0`
    3. EMA crossover: `row[ema_short_col] > row[ema_long_col]`
    4. Volume above 20-bar average: `row["volume"] > df["volume"].rolling(20).mean().iloc[-1]`
  - **SELL conditions** (any true):
    1. RSI > 70 (overbought) OR MACD histogram turns negative
    2. EMA 9 crosses below EMA 21
  - Confidence: 0.8 if all BUY conditions met, 0.6 if SELL conditions met
  - `atr = row[atr_col]`; `stop_price = round(row["close"] - (atr * params.get("atr_multiplier", 1.5)), 2)`
  - Return `Signal(action="BUY"|"SELL"|"HOLD", confidence=..., symbol=symbol, strategy="momentum", atr=atr, stop_price=stop_price, reasoning="...")`

**`scripts/strategies/mean_reversion.py` — MeanReversionStrategy (STRAT-02):**
- **BUY conditions** (all must be true):
  1. Price at or below lower Bollinger Band: `row["close"] <= row[bb_lower_col]`
  2. RSI below 30: `row[rsi_col] < 30`
  3. Price within 2% of lower band: `abs(row["close"] - row[bb_lower_col]) / row[bb_lower_col] <= 0.02`
- **SELL conditions**:
  1. Price returns to middle band: `row["close"] >= row[bb_middle_col]`
- Stop: ATR-based. Confidence: 0.75 for BUY, 0.7 for SELL.

**`scripts/strategies/breakout.py` — BreakoutStrategy (STRAT-03):**
- **BUY conditions** (all must be true):
  1. New 20-bar high: `row["close"] > df["high"].rolling(20).max().iloc[-2]` (compare against prior bar's rolling max to confirm breakout)
  2. Volume > 1.5x 20-bar average: `row["volume"] > df["volume"].rolling(20).mean().iloc[-1] * 1.5`
- **SELL conditions**:
  1. Price falls below the 20-bar high (failed continuation)
- Stop: ATR-based. Confidence: 0.7 for BUY.

**`scripts/strategies/vwap.py` — VWAPStrategy (STRAT-04):**
- **BUY conditions** (all must be true):
  1. Price > 1.5% below VWAP: `(row["VWAP_D"] - row["close"]) / row["VWAP_D"] > 0.015`
  2. RSI < 40: `row[rsi_col] < 40`
  3. Time between 10:00-15:00 ET: check `df.index[-1].hour` is between 10 and 14 (inclusive)
- **SELL conditions**:
  1. Price returns to VWAP: `row["close"] >= row["VWAP_D"]`
- Stop: percentage-based (not ATR): `stop_price = round(row["close"] * (1 - params.get("max_deviation_pct", 3.0) / 100), 2)`. ATR field still populated for logging.
- Confidence: 0.7 for BUY.

All strategies: use loguru for logging. Handle missing columns gracefully (return HOLD). Import Signal from `scripts.types`.
