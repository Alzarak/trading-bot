```python
from dataclasses import dataclass
from typing import Literal

@dataclass
class Signal:
    action: Literal["BUY", "SELL", "HOLD"]
    confidence: float       # 0.0 - 1.0
    symbol: str
    strategy: str
    atr: float              # Current ATR value for stop calculation
    stop_price: float       # Pre-computed stop price
    reasoning: str          # Audit log text
```

Create `scripts/market_scanner.py` with a `MarketScanner` class:

1. `__init__(self, trading_client, data_client, config: dict)` — stores clients and config. `trading_client` is `TradingClient`, `data_client` is `StockHistoricalDataClient`. Extract `strategy_params` from config (rsi_period=14, macd_fast=12, macd_slow=26, macd_signal=9, ema_short=9, ema_long=21, atr_period=14, bb_period=20, bb_std_dev=2.0) with defaults.

2. `compute_indicators(self, df: pd.DataFrame) -> pd.DataFrame` — appends all 6 indicators using pandas-ta:
   - `df.ta.rsi(length=rsi_period, append=True)` -> column `RSI_{rsi_period}`
   - `df.ta.macd(fast=macd_fast, slow=macd_slow, signal=macd_signal, append=True)` -> columns `MACD_{fast}_{slow}_{signal}`, `MACDh_{fast}_{slow}_{signal}`, `MACDs_{fast}_{slow}_{signal}`
   - `df.ta.ema(length=ema_short, append=True)` and `df.ta.ema(length=ema_long, append=True)` -> columns `EMA_{short}`, `EMA_{long}`
   - `df.ta.atr(length=atr_period, append=True)` -> column `ATRr_{atr_period}`
   - `df.ta.bbands(length=bb_period, std=bb_std_dev, append=True)` -> columns `BBL_{bb_period}_{bb_std_dev}`, `BBM_{bb_period}_{bb_std_dev}`, `BBU_{bb_period}_{bb_std_dev}`
   - `df.ta.vwap(anchor="D", append=True)` -> column `VWAP_D`
   - CRITICAL: Before calling `df.ta.vwap()`, ensure `df.index` is a timezone-aware DatetimeIndex. If `df.index.tz is None`, localize to UTC then convert to `America/New_York`. If already UTC, just convert.
   - After all indicators, call `df.dropna()` to handle IEX feed gaps and rolling window warmup NaN values.
   - Return the cleaned DataFrame.

3. `get_indicator_columns(self) -> dict[str, str]` — returns a mapping of logical names to actual column names derived from params. Example: `{"rsi": "RSI_14", "atr": "ATRr_14", "macd": "MACD_12_26_9", "macd_histogram": "MACDh_12_26_9", "macd_signal": "MACDs_12_26_9", "ema_short": "EMA_9", "ema_long": "EMA_21", "bb_lower": "BBL_20_2.0", "bb_middle": "BBM_20_2.0", "bb_upper": "BBU_20_2.0", "vwap": "VWAP_D"}`. This prevents the ATR column name mismatch pitfall (ATRr_14 not ATR_14).

4. `fetch_bars(self, symbol: str, days_back: int = 60) -> pd.DataFrame` — fetches OHLCV bars from Alpaca using `StockHistoricalDataClient.get_stock_bars()`:
   - Build `StockBarsRequest(symbol_or_symbols=symbol, timeframe=TimeFrame.Minute, start=datetime.now(ET) - timedelta(days=days_back), end=datetime.now(ET), feed="iex")`
   - `bars = self.data_client.get_stock_bars(request)`
   - `df = bars.df.reset_index(level=0, drop=True)` — removes multi-level symbol index
   - Timezone handling: if `df.index.tz is None`, localize to UTC; then convert to `America/New_York` via `df.index.tz_convert("America/New_York")`
   - Return df

5. `scan(self, symbol: str) -> pd.DataFrame` — combines fetch + compute: calls `fetch_bars(symbol)` then `compute_indicators(df)`. Returns indicator-enriched, NaN-free DataFrame. Returns empty DataFrame if bars are insufficient.

6. `is_market_open(self) -> bool` — calls `self.trading_client.get_clock()` and returns `clock.is_open`.

Use `from zoneinfo import ZoneInfo` for timezone. Define `ET = ZoneInfo("America/New_York")` at module level. Use loguru for logging. Import alpaca-py conditionally (try/except) like risk_manager.py does for testability.
